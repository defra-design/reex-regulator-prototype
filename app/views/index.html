Updates based on feedback from user research and stakeholder review, including:

  • submission date added to card for applications that have not been duly made
  • organisation name more prominent on cards
  • duly making process updated to start from application summary, which:
    • gives access to other actions like query and withdraw
    • means the application has to be manually moved to in progress, either when assigning to yourself or another, or if assigned to you and still in duly making
  • adding an option to unassign a case
  • withdraw option added to applications
  • changing language from determination to decision
  • removing rejected option from decisions
  • added an example where no applications are found (use apply filters to view again and clear filters to remove)
