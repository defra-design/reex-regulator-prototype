# REEX regulator prototype

Prototypes of the regulator case working process built using the GOV.UK Prototype Kit. Go to the [GOV.UK Prototype Kit site](https://prototype-kit.service.gov.uk/docs) to read the full documentation.

## Requirements

Node.js LTS version 20.x.x

## Running the prototype locally

Open a terminal at your project and run:
- `npm install` if this is your first time running the prototype
- `npm run dev` to start the prototype

Go to localhost:3000 in your browser.
